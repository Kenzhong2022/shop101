# 首页性能优化详解

## 1. 图片加载优化

### Cloudinary 图片组件

使用自定义 `kk-cld-image.vue` 组件实现云图片优化：

- 使用 `loading="lazy"` 实现图片懒加载
- 首页轮播图设置 `:priority="true"` 优先加载首屏关键图片
- 自动拼接 CDN 前缀，支持质量压缩

```vue
<kk-cld-image
  :src="item.image"
  :alt="item.title"
  :width="700"
  :height="450"
  gravity="auto"
  crop="fill"
  :priority="true"
/>
```

### Nuxt Image 全局配置

在 `nuxt.config.js` 中配置全局图片优化策略：

```javascript
image: {
  quality: 2,           // 高质量压缩
  format: ["webp"],     // 优先使用 WebP 格式
  lazy: true,           // 全局懒加载
  preload: true,        // 预加载配置
}
```

## 2. 构建时图片压缩优化

使用 `vite-plugin-image-optimizer` 插件在构建阶段压缩图片体积：

```javascript
vite: {
  plugins: [
    ViteImageOptimizer({
      png: {
        quality: 2,
        speed: 4,
      },
      jpeg: {
        quality: 2,
        progressive: true, // 渐进式加载
      },
      webp: {
        quality: 2,
        lossless: false,
      },
      avif: {
        quality: 2,
        lossless: false,
      },
      svg: {
        multipass: true,
        plugins: [
          { name: "removeEmptyAttrs", active: true },
        ],
      },
      gif: {
        optimizationLevel: 3,
      },
    }),
  ],
}
```

## 3. 首屏性能优化

首页文件明确标注已优化首次加载性能：

```html
<!-- 这是首页页面 - 已优化首次加载性能 -->
```

具体措施：

- 轮播图使用 `type="card"` 配合固定 `aspectRatio: '7/2'`
- 商品列表采用**分页加载**机制（每页 5 条）
- 通过 `loadMore` 函数实现滚动加载更多，减少初始请求量

```typescript
async function getGoodsList() {
  await apiGoodsList({
    page: currentPage.value,
    page_size: 5,
  }).then((res) => {
    hasMore.value = res.data.list.length !== 0;
    goodsList.value = [...(goodsList.value || []), ...res.data.list];
  });
}
```

## 4. 数据获取优化

- 使用 `useLazyAsyncData` 实现非阻塞式数据获取
- 商品列表支持追加模式，避免全量数据替换
- 配合 Loading 状态管理，提升用户体验

```typescript
const loadingStore = useLoadingStore();
loadingGoods.value = true;
await apiGoodsList({...}).finally(() => {
  loadingGoods.value = false;
});
```

## 5. 组件渲染优化

在用户中心等页面使用的优化策略：

- `markRaw()` 包裹异步组件，避免被 Vue 响应式系统代理
- `defineAsyncComponent()` 异步加载组件，按需加载
- `shallowRef` 优化大数据场景的响应式开销

```typescript
const MyUserInfo = markRaw(
  defineAsyncComponent(() => import("./myUserInfo.vue")),
);
const currentComponent = shallowRef(null);
```

## 6. 布局与样式优化

- 响应式设计：使用 `@media` 适配不同屏幕尺寸
- 轮播图 `transform: scale` 悬停动画增强交互体验
- 使用 CSS 变量和 Tailwind 原子化 CSS 减少样式体积
- 移除生产环境 `console.*` 和 `debugger` 调试代码

```javascript
terserOptions: {
  compress: {
    drop_console: true,
    drop_debugger: true,
  },
}
```
