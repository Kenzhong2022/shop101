## 1.1.3 数据模型

### 核心类型定义

```typescript
// server/types/goods-specs.ts

/** 规格值 */
interface SpecValue {
  id: string;
  name: string; // 显示名：中国红
  code: string; // 代码：red
}

/** 规格维度（如颜色、尺码） */
interface SpecDimension {
  id: string;
  name: string; // 显示名：颜色
  code: string; // 字段名：color
  sort_order: number;
  values: SpecValue[];
}

/** SKU 库存单元 */
interface SkuInfo {
  sku_id: string;
  sku_code: string; // 商家编码：RED-S-COT
  specs_hash: string; // 规格组合：red|s|cotton
  specs: string[]; // 规格值数组：['red', 's', 'cotton']
  price: string;
  stock: number;
}

/** 商品规格数据 */
interface GoodsSpecData {
  goods_id: number;
  goods_name: string;
  image: string;
  base_price: string;
  has_spec: boolean;
  dimensions: SpecDimension[];
  skus: SkuInfo[];
}
```

## 1.1.4 后端 API

### 获取商品规格信息

**接口**: `GET /api/goods/:id/specs`

**核心实现** (`server/api/goods/[id]/specs.get.ts`):

```typescript
export default defineEventHandler(async (event) => {
  const goodsId = getRouterParam(event, "id");

  // 并行查询商品信息、规格维度、SKU
  const [goods, dimensions, skus] = await Promise.all([
    mySql`
      SELECT id, goods_name, image, price
      FROM goods WHERE id = ${id}
    `,
    mySql`
      SELECT id, name, code, sort_order
      FROM spec_dimensions
      WHERE goods_id = ${id}
      ORDER BY sort_order ASC
    `,
    mySql`
      SELECT id, sku_code, price, stock, specs_hash,
             string_to_array(specs_hash, '|') as specs
      FROM skus WHERE goods_id = ${id}
    `,
  ]);

  return {
    code: 200,
    data: {
      goods_id: id,
      goods_name: goods[0].goods_name,
      image: goods[0].image,
      base_price: goods[0].price,
      has_spec: dimensions.length > 0,
      dimensions: processDimensions(dimensions),
      skus,
    },
  };
});
```

### goods-aside.vue 规格选择核心逻辑

规格选择面板是商品详情页的核心组件，实现了**动态联动禁用算法**。

#### 1. 数据映射

```typescript
// 维度 ID 数组
const specKeys = computed(() => props.specList.map((spec) => spec.code));

// SKU 查找映射表：specs_hash -> SkuInfo
const findGoodsBySpecsHash = computed(() => {
  const map = new Map<string, SkuInfo>();
  props.skuList.forEach((sku) => {
    map.set(sku.specs_hash, sku);
  });
  return map;
});

// 规格值显示名映射
const valueNameMap = computed(() => {
  const map = new Map<string, string>();
  props.specList.forEach((spec) => {
    spec.values.forEach((val) => {
      map.set(`${spec.code}:${val.code}`, val.name);
    });
  });
  return map;
});
```

#### 2. 联动禁用算法

**核心函数**: `isSpecDisabled(dimId, valueId)`

判断某个规格值是否应该被禁用。逻辑是：假设用户选中该值后，检查是否存在任何有效 SKU（库存 > 0）满足所有已选规格。

```typescript
function isSpecDisabled(dimId: string, valueId: string): boolean {
  // 已选中的不禁用（允许取消）
  if (isSelected(dimId, valueId)) return false;

  // 构造假设的选中状态
  const assumedSelection = { ...selected, [dimId]: valueId };

  // 检查是否存在有效 SKU
  const hasValidSku = props.skuList.some((sku) => {
    const isMatch = specKeys.value.every((key, index) => {
      const selectedVal = assumedSelection[key];
      if (selectedVal === undefined) return true; // 未选规格通配
      return sku.specs[index] === selectedVal;
    });
    return isMatch && sku.stock > 0;
  });

  return !hasValidSku;
}
```

#### 3. 选中状态管理

```typescript
const selected = reactive<Record<string, string>>({});

/** 是否选完所有维度 */
const isComplete = computed(() => selectedCount.value === dimensionCount.value);

/** 当前 SKU Key（如 "red|l|cotton"）*/
const currentSkuKey = computed(() => {
  if (!isComplete.value) return "";
  return specKeys.value.map((key) => selected[key]).join("|");
});

/** 当前选中 SKU 的显示名称（如 "中国红 / L / 纯棉"）*/
const currentSkuDisplayName = computed(() => {
  if (!isComplete.value) return "";
  return specKeys.value
    .map((dimId) => {
      const valCode = selected[dimId];
      return valueNameMap.value.get(`${dimId}:${valCode}`) || valCode;
    })
    .join(" / ");
});

/** 当前选中的完整 SKU 信息 */
const currentSku = computed(() => {
  if (!isComplete.value) return null;
  return findGoodsBySpecsHash.value.get(currentSkuKey.value);
});
```

#### 4. 按钮样式

```typescript
function getButtonClasses(dimId: string, valueId: string): string {
  const selected = isSelected(dimId, valueId);
  const disabled = isSpecDisabled(dimId, valueId);

  if (disabled && !selected) {
    return "bg-#ccc text-gray-400 border-#ccc cursor-not-allowed";
  }
  if (selected) {
    return "border-primary border-2 text-primary bg-#fff transform scale-115";
  }
  return "hover:border-primary hover:text-primary active:scale-95 bg-#fff border-#ccc";
}
```

## 1.1.6 页面布局定位

右侧规格面板采用**固定定位**，确保滚动时始终可见：

```vue
<goods-aside
  class="fixed left-[calc(50%+800px-600px/2-16px-24px)] -translate-x-1/2
         w-[600px] z-999"
  :style="{
    top: (scrollY < 65 ? 65 - Math.floor(scrollY) : 0) + 'px',
  }"
/>
```

计算逻辑：

- `50%` 页面中心
- `+800px` 右侧偏移到内容区
- `-600px/2` 面板宽度一半
- `-16px-24px` 间距调整

## 1.1.7 用户行为追踪

商品详情页实现了**浏览时长统计**功能：

```typescript
let durationInterval: any = null;
let sessionStartTime: number | null = null;
let totalDuration = 0;

const startTimer = () => {
  sessionStartTime = Date.now();
  durationInterval = setInterval(() => {
    console.log("已浏览:", getCurrentDuration() / 1000, "秒");
  }, 1000);
};

const stopTimer = () => {
  if (durationInterval) {
    clearInterval(durationInterval);
    durationInterval = null;
  }
  if (sessionStartTime) {
    totalDuration += Date.now() - sessionStartTime;
    sessionStartTime = null;
  }
};

onActivated(() => startTimer());
onDeactivated(() => {
  stopTimer();
  track(); // 上报累计浏览时长
});
```
